
# Generating test data 

## Usage

```
Usage of /Users/ralf/go/bin/gen:
  -8	flip all 8 bits of a single byte
  -Version
    	V0.1.728
  -n int
    	number of samples to write (default 4)
  -s int
    	size of data file to create (default 1000)
  -sha
    	write .sha512 file containing hex-digest
```
